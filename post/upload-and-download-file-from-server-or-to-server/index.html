It's my first time to use linux in this semester, I have many questions to solve, keep this tag as a tip for myself.

Upload and download files or directories from server and to server.

Local to server

  • First thing first, login in to the server:
    sftp <user_name>@XX.XXX.XXX.XX
    enter the password of server.

Upload file to the server

>sftp put <path of local document> <path on server>
In this part, should pay more attention on the path of local, the path would comes from C:/... NOT FROM C:/cygwin/

Upload directory to the server

>sftp put -r <path of local directory> <path on server>

Server to local

Download file to local

>sftp get <path of server document> <path on local>

Download directory to local

>sftp get -r <path of server directory> <path on local>

Other command of sftp

lcd: Enter the path of local server
cd: Enter the path of cloud server
pwd: Query the path of this document/directory
lpwd: Query the path of local

Most of the origanial comment could be used in the sftp mode. The local ones need to +l
